# Servant
[![image](https://www.servant.dev/images/servant.png)](https://www.servant.dev/)
- haskell web framework
- [GitHub](https://github.com/haskell-servant/servant)

## Install servant project
```shell
 $ stack new name-project servant
```
## Build stack
```shell
 $ stack build
```
## Run service
```shell
 $ stack ghci
 Prelude> main
```
